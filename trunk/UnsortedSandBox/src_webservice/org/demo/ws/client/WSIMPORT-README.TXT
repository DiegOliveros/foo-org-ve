¡Todo este código fue generado con wsimport!

El cliente fue generado con:

wsimport -Xnocompile -s . -p org.demo.ws.client http://localhost:8080/WS/HelloWorld?wsdl

El comando tiene que ejecutarse desde DemoWebService/src, y tiene que estar
corriendo org.demo.ws.server.WSPublisher para que wsimport pueda encontrar el
wsdl. En cualquier caso, al abrir en un navegador el URL
http://localhost:8080/WS/HelloWorld?wsdl se debe de ver el wsdl.
